You can select one of the following commands: 
  1) PLACE X, Y, F 
  which will put the toy robot on the table in position X,Y
  and facing NORTH, SOUTH, EAST or WEST. 
  2) MOVE 
  which will move the toy robot one unit forward in the direction it is currently
  facing. 
  3) LEFT 
  which will rotate the robot 90 degrees in the left direction. 
  4) RIGHT 
  which will rotate the robot 90 degrees in the right direction. 
  5) REPORT 
  which will announce the X,Y and F of the robot.
